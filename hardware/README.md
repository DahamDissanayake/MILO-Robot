# hardware — mechanical and firmware reference

This folder holds hardware-facing reference material for Milo — not code
that runs on the robot itself (that's [`bridge/`](../bridge/)).

## What's in here

```
hardware/
  reference-sesame/   original Sesame Robot Project firmware files Milo
                       ports from (servo angles, face bitmaps) — see its
                       own README for what each file maps to
```
